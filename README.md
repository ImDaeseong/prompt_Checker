# Prompt Checker

ChatGPT 또는 Gemini에서 내 프롬프트를 바로 테스트하고, 더 나은 프롬프트로 고쳐주는 Chrome 확장 프로그램입니다.

서버가 필요 없습니다. API 키도 필요 없습니다.  
이미 열어둔 ChatGPT/Gemini 탭을 이용합니다.

## 이런 분에게 좋습니다

- 내가 만든 프롬프트가 좋은지 빠르게 확인하고 싶은 분
- ChatGPT/Gemini에게 프롬프트 개선을 반복해서 맡기는 분
- 프롬프트 수정 전후를 비교하고 싶은 분
- 별도 서버나 복잡한 설치 없이 가볍게 쓰고 싶은 분

## 설치 방법

### 1. 다운로드

이 페이지에서 초록색 **Code** 버튼을 누른 뒤 **Download ZIP**을 클릭합니다.

다운로드한 ZIP 파일의 압축을 풉니다.

### 2. Chrome 확장 프로그램 화면 열기

Chrome 주소창에 아래 주소를 입력합니다.

```text
chrome://extensions
```

오른쪽 위의 **개발자 모드**를 켭니다.

### 3. 확장 프로그램 불러오기

**압축해제된 확장 프로그램을 로드** 버튼을 누릅니다.

압축을 푼 폴더 안에서 아래 폴더를 선택합니다.

```text
extension
```

설치가 끝나면 Chrome 툴바에 Prompt Checker 아이콘이 나타납니다.

## 사용 방법

1. ChatGPT 또는 Gemini를 Chrome에서 엽니다.
2. 이미 열려 있던 탭이라면 한 번 새로고침합니다.
3. Chrome 툴바에서 **Prompt Checker** 아이콘을 누릅니다.
4. **내가 만든 프롬프트**에 검사할 프롬프트를 붙여넣습니다.
5. **테스트 질문**에 그 프롬프트로 물어볼 예시 질문을 입력합니다.
6. 필요하면 **개선 방향**에 원하는 기준을 적습니다.
7. **테스트 + 평가 실행**을 누릅니다.

Prompt Checker가 ChatGPT/Gemini 탭에 내용을 자동 입력하고, 새로 나온 답변을 가져옵니다.

결과가 마음에 들면 **업데이트** 버튼을 눌러 개선된 프롬프트를 입력란에 반영할 수 있습니다.

## 입력 예시

### 내가 만든 프롬프트

```text
너는 친절한 코딩 도우미다. 사용자의 질문에 답해라.
```

### 테스트 질문

```text
파이썬에서 파일을 한 줄씩 읽는 예제를 알려줘.
```

### 개선 방향

```text
초보자도 이해할 수 있게 설명하고, 짧은 코드 예제를 포함해줘.
```

## 자주 생기는 문제

### 대상 탭이 없다고 나옵니다

ChatGPT 또는 Gemini 탭을 열고 새로고침한 뒤 다시 시도하세요.

지원 주소:

```text
https://chatgpt.com
https://chat.openai.com
https://gemini.google.com
```

### 입력창을 찾을 수 없다고 나옵니다

ChatGPT/Gemini 화면이 완전히 로딩된 뒤 다시 실행하세요.  
그래도 안 되면 해당 탭을 새로고침하세요.

### 결과가 오래 걸립니다

AI 답변 생성이 끝날 때까지 기다립니다.  
5분이 지나면 자동으로 중단됩니다.

### 업데이트 버튼은 무엇인가요?

AI가 만들어준 **개선된 프롬프트**를 다시 입력란에 넣는 버튼입니다.  
마음에 드는 결과가 나오면 업데이트한 뒤 다시 테스트할 수 있습니다.

## 개인정보와 보안

- 이 확장 프로그램은 별도 서버를 사용하지 않습니다.
- API 키, 토큰, 비밀번호를 요구하지 않습니다.
- 입력한 내용은 사용자가 열어둔 ChatGPT/Gemini 탭으로 전송됩니다.
- 회사 내부 정보, 고객 정보, 비밀번호, API 키, 토큰은 입력하지 마세요.
- ChatGPT/Gemini에 입력하면 안 되는 민감한 내용은 이 확장 프로그램에도 입력하면 안 됩니다.

## 필요한 권한

- `tabs`: 열려 있는 ChatGPT/Gemini 탭을 찾기 위해 사용합니다.
- `sidePanel`: Chrome 사이드 패널 화면을 열기 위해 사용합니다.
- `windows`: 현재 브라우저 창에서 패널 상태를 관리하기 위해 사용합니다.

## 제한 사항

- ChatGPT/Gemini 화면 구조가 바뀌면 일시적으로 동작하지 않을 수 있습니다.
- Chrome 확장 프로그램입니다. 다른 브라우저에서는 테스트하지 않았습니다.
- 답변 품질은 사용하는 AI 모델과 입력한 테스트 질문에 따라 달라집니다.

## 개발자용

Chrome Web Store에 직접 올릴 때는 `extension` 폴더 내용을 패키징하면 됩니다.  
ZIP 파일 안에서 `manifest.json`이 최상위에 있어야 합니다.

파일 구조:

```text
extension/
├── icons/
├── manifest.json
├── background.js
├── popup.html
├── popup.js
└── content.js
```
